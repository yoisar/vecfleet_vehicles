# Vehicle management web application
## Web application that allows vehicle management. 

## Vehicle data : 
 - Vehicle type 
 - Wheels 
 - Brand 
 - Model 
 - Patent 
 - Chassis number 
 - Kms traveled 

## Modules: 
 - List of vehicles 
 - Add vehicle 
 - Delete vehicle 
 - Search vehicle 
 - Edit vehicle

## Technologies / Frantend Frameworks: 
 - React (Hooks) 
 - Bootstrap 
 - Responsive Design 
 - Javascript 
 
## BACKEND: 
 - ORM (Eloquent, doctrine, others) 
 - MySQL / PostgreSql database 
 - API rest.
